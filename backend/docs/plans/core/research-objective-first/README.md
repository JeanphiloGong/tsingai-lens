# Research Objective First

This topic family records the Core plan for moving semantic extraction from a
paper-wide or material-wide scan into objective-shaped research workspaces.

The family keeps multiple live pages because this local subject now needs:

- a parent flow plan that explains why Core first discovers multiple research
  objectives from a collection
- a domain-model slice that adds the pure Core records before service and
  prompt work
- an objective-context extraction slice that threads the selected objective
  into paper-fact routing and extraction
- a P001 gold-gap repair slice that uses the objective-shaped extraction result
  to finish method-family conditions, pairwise comparison relations, and
  characterization observations

This is a Core semantic-build topic family. It does not change public API
routes, frontend contracts, Source artifact production, or downstream report
ownership by itself.
